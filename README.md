## Key Features

- <img src="https://api.iconify.design/lucide:zap.svg?color=%23f59e0b" width="20" height="20" align="center" /> **Next.js & React 19**: Built on the latest foundations for ultimate performance.
- <img src="https://api.iconify.design/lucide:palette.svg?color=%23ec4899" width="20" height="20" align="center" /> **Modern Design**: Premium, aesthetically pleasing components out of the box.
- <img src="https://api.iconify.design/lucide:layers.svg?color=%238b5cf6" width="20" height="20" align="center" /> **Flexible & Modular**: Highly customizable components using Radix UI and Tailwind CSS.
- <img src="https://api.iconify.design/lucide:book-open.svg?color=%233b82f6" width="20" height="20" align="center" /> **Documentation Ready**: Integrated with **Fumadocs** for beautiful, searchable docs.
- <img src="https://api.iconify.design/lucide:moon.svg?color=%2364748b" width="20" height="20" align="center" /> **Dark Mode Support**: Seamless theme switching with `next-themes`.
- <img src="https://api.iconify.design/lucide:activity.svg?color=%23ef4444" width="20" height="20" align="center" /> **Smooth Animations**: Powered by **Framer Motion** for a dynamic user experience.

## <img src="https://api.iconify.design/lucide:rocket.svg?color=%23f97316" width="24" height="24" align="center" /> Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/bilals2008/bilal-ui.git
cd bilal-ui
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## <img src="https://api.iconify.design/lucide:folder-tree.svg?color=%2310b981" width="24" height="24" align="center" /> Project Structure

```bash
├── app/              # Next.js App Router (Pages & Layouts)
├── components/       # Custom UI components
│   ├── bilalUi/      # Bilal UI specific components
│   ├── landing/      # Landing page sections
│   ├── ui/           # Base UI primitives (Radix-based)
│   └── icons/        # Custom icons set
├── content/          # Documentation MDX files
├── lib/              # Shared utilities and configurations
├── public/           # Static assets (images, logos)
└── scripts/          # Helper scripts
```

## <img src="https://api.iconify.design/lucide:file-text.svg?color=%233b82f6" width="24" height="24" align="center" /> Documentation

The documentation is located in the `content/docs` directory and is served via the `/docs` route. You can easily add or modify documentation by editing the MDX files.
