# Button

Displays a button or a component that looks like a button.

## Usage

<ComponentPreview name="button-demo" code={`import { Button } from "@/components/ui/button"

  return <Button>Button</Button>
}`}>
  <Button>Button</Button>
</ComponentPreview>
