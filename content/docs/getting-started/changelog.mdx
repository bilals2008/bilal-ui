## v0.1.0-alpha
*January 2026*

The first public look at Bilal UI. We've laid the foundation for a premium, copy-paste component library.

### Core Foundation
- **Tailwind CSS 4**: Full support for the latest CSS engine.
- **Next.js 16**: Built for the latest app router features.
- **Framer Motion Integration**: Smooth transitions out of the box.

### Initial Components
- **Navigation**: Premium Sidebar and Table of Contents with gradient indicators.
- **Layout**: High-performance documentation layout using Fumadocs.
- **UI Kit**: Core Radix-based components (Buttons, Cards, Inputs).

---

### What's Next?
- More animated specialized components.
- Enhanced dark mode refinements.
- Interactive playground for component customization.
