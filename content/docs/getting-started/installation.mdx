## Prerequisites

Before installing Bilal UI, ensure your project meets the following requirements:

- **React** 18 or higher
- **Tailwind CSS** v4
- **TypeScript** 5+ (recommended)

<Callout type="info">
  All components are written in **TypeScript**. While you can use them in JavaScript projects, TypeScript is recommended for the best developer experience.
</Callout>

## 1. Install utilities

All components use [Tailwind CSS v4](https://tailwindcss.com), so ensure it's installed in your project. Many components also use the `cn` utility function—you can set this up by following the framework guides below:

<br />
<Callout type="success">
  If you already have a project compatible with Shadcn, you can skip this step.
</Callout>

## 2. That's it

We use [lucide-icons](https://lucide.dev) for most components that include icons, along with some [shadcn/ui](https://ui.shadcn.com) components. These dependencies will be automatically installed when using the CLI.

For example to add `button01` to your project, it will be easy as:

<Snippet defaultValue="bun">
  <SnippetHeader>
    <SnippetTabsList>
       <SnippetTabsTrigger value="bun">bun</SnippetTabsTrigger>
       <SnippetTabsTrigger value="npx">npx</SnippetTabsTrigger>
       <SnippetTabsTrigger value="pnpm">pnpm</SnippetTabsTrigger>
    </SnippetTabsList>
    <SnippetCopyButton value="bunx shadcn@latest add https://ui.bilal.dev/api/source/button01.json" />
  </SnippetHeader>
  <SnippetTabsContent value="bun">
  <span className="text-pink-500">bunx </span>shadcn@latest add https://ui.bilal.dev/api/source/button01.json
  </SnippetTabsContent>
  <SnippetTabsContent value="npx">
  <span className="text-pink-500">npx </span>shadcn@latest add https://ui.bilal.dev/api/source/button01.json
  </SnippetTabsContent>
  <SnippetTabsContent value="pnpm">
  <span className="text-pink-500">pnpm dlx </span>shadcn@latest add https://ui.bilal.dev/api/source/button01.json
  </SnippetTabsContent>
</Snippet>
<br />
<Callout type="tip">
  While we provide a convenient 'copy' button for the code, we strongly recommend using the CLI for installing components, as it ensures all necessary files are included.
</Callout>

Add to your page and it works!

```tsx
import { MagneticButton } from "@/components/bilalUi/button/Button01";

export default function Page() {
  return <MagneticButton />;
}
```
