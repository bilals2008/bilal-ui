## Prerequisites

Before you can install and use Bilal UI, make sure your project meets the following requirements:

- **React**: Version 19 or higher.
- **Typescript**: Version 5 or higher.
- **Tailwind CSS**: Version 3.4 or higher.
- **Radix UI**: Version 1 or higher.
- **Base UI**: Version 1 or higher.
- **Motion**: Version 12.0 or higher.

<Step>
### Setup Inter Font (Next.js)

Add the following code to your root layout file (`app/layout.tsx` or `app/layout.js`):

```tsx
import { Inter } from "next/font/google";
import { cn } from "@/lib/utils";

const inter = Inter({ subsets: ["latin"] });

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body className={cn("antialiased", inter.className)}>{children}</body>
    </html>
  );
}
```

</Step>

<Step>
### Setup System Fonts

Add the following code to your style entry file `globals.css`:

```css
@theme {
  --font-sans:
    "Inter", "Inter Fallback", ui-sans-serif, system-ui, sans-serif,
    "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Noto Color Emoji";
  --font-mono: ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New";
}
```

<Callout type="tip">
For a modern and visually appealing design, we recommend using Inter as the default font in your project.
</Callout>

</Step>
