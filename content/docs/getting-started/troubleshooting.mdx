## Common Issues

<div className="space-y-6">

### <AlertCircle className="inline-block w-4 h-4 mr-2 text-red-500" /> Component not found error?

- Ensure the component name matches exactly (case-insensitive)
- Check that your project has `components.json` configured

### <Paintbrush className="inline-block w-4 h-4 mr-2 text-blue-500" /> Styles not applying?

- Verify Tailwind CSS v4 is properly configured
- Check that `@tailwind` directives are in your CSS entry file

### <Code className="inline-block w-4 h-4 mr-2 text-purple-500" /> TypeScript errors?

- Ensure TypeScript 5+ is installed
- Run `npx tsc --noEmit` to check for type issues

</div>
