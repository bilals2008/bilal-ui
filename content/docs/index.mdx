## Getting Started

Follow these steps to integrate Bilal UI into your project.

### 1. Prerequisites

Ensure you have the following dependencies installed in your project:

```bash
npm install lucide-react clsx tailwind-merge
```

### 2. Configure Utilities

Add the following `cn` helper to your `lib/utils.ts` file to handle Tailwind class merging:

```typescript
import { clsx, type ClassValue } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

### 3. Usage

Browse the components in the sidebar, choose one, and follow the specific instructions on its page. Most components are designed to be "grab and go."
